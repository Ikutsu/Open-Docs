# 예제

Koog 프레임워크는 다양한 사용 사례에 맞는 AI 에이전트를 구현하는 방법을 이해하는 데 도움이 되는 예제를 제공합니다. 이 예제들은 여러분이 자신의 애플리케이션에 적용할 수 있는 주요 기능과 패턴을 보여줍니다.

아래 예제를 찾아보고 링크를 클릭하여 GitHub에서 소스 코드를 확인하세요.

| 예제                                                                                                                             | 설명                                                                                                                                                                                                                                                      |
|-------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Attachments](examples/Attachments.md)                                                                                              | 구조화된 마크다운(Markdown)과 첨부 파일(attachments)을 프롬프트에서 사용하는 방법을 알아봅니다. 이미지를 포함하고 OpenAI 모델을 사용하여 Instagram 게시물용 창의적인 콘텐츠를 생성하는 프롬프트를 구축합니다.                                                                                        |
| [Banking](examples/Banking.md)                                                                                                      | 정교한 그래프 기반 전략을 통해 송금 및 거래 분석을 처리할 수 있는 라우팅 기능을 갖춘 포괄적인 AI 뱅킹 어시스턴트를 구축합니다. 도메인 모델링, 도구 생성, 에이전트 구성 패턴을 포함합니다.             |
| [BedrockAgent](examples/BedrockAgent.md)                                                                                            | AWS Bedrock 통합을 통해 Koog 프레임워크를 사용하여 지능형 AI 에이전트를 생성합니다. 사용자 지정 도구를 정의하고 AWS Bedrock을 설정하며, 장치 제어를 위한 자연어 명령을 이해하는 대화형 에이전트를 구축하는 방법을 알아봅니다.                        |
| [Calculator](examples/Calculator.md)                                                                                                | 덧셈, 뺄셈, 곱셈, 나눗셈 도구를 사용하여 산술 연산을 수행하는 계산기 에이전트를 구축합니다. 병렬 도구 호출, 이벤트 로깅 및 다중 실행기 지원(OpenAI 및 Ollama)을 보여줍니다.                            |
| [Chess](examples/Chess.md)                                                                                                          | 복잡한 도메인 모델링, 사용자 지정 도구, 메모리 최적화 기술 및 대화형 선택 기능을 갖춘 지능형 체스 플레이 에이전트를 구축합니다. 고급 에이전트 전략, 게임 상태 관리, 인간-AI 협업 패턴을 보여줍니다.  |
| [GoogleMapsMcp](examples/GoogleMapsMcp.md)                                                                                          | Docker를 통해 Koog를 Google Maps MCP 서버에 연결합니다. Kotlin Notebook 환경에서 AI 에이전트를 사용하여 실제 지리 API로 도구를 검색하고, 주소를 지오코딩하며, 고도 데이터를 가져오는 방법을 알아봅니다.                                                               |
| [Guesser](examples/Guesser.md)                                                                                                      | 목표 질문을 하기 위한 도구를 사용하여 이진 탐색 전략을 구현하는 숫자 맞추기 에이전트를 구축합니다. 에이전트는 전략적인 질문을 통해 사용자의 숫자를 효율적으로 좁히고 도구 기반 상호작용 패턴을 보여줍니다.               |
| [Langfuse](examples/Langfuse.md)                                                                                                    | OpenTelemetry를 사용하여 Koog 에이전트 트레이스(traces)를 Langfuse로 내보내는 방법을 알아봅니다. 환경 변수를 설정하고 에이전트를 실행하며, 포괄적인 가시성(observability)을 위해 Langfuse 인스턴스에서 스팬(spans) 및 트레이스(traces)를 검사합니다.                                                         |
| [MCP](https://github.com/JetBrains/koog/tree/develop/examples/src/main/kotlin/ai/koog/agents/example/mcp)                           | 지리 데이터용 GoogleMapsMcpClient와 브라우저 자동화용 PlaywrightMcpClient를 특징으로 하는 모델 컨텍스트 프로토콜(Model Context Protocol) 통합 예제입니다.                                                                                                           |
| [Memory](https://github.com/JetBrains/koog/tree/develop/examples/src/main/kotlin/ai/koog/agents/example/memory)                     | 메모리 시스템 사용법을 보여주는 고객 지원 에이전트입니다. 이 에이전트는 암호화된 로컬 스토리지를 사용하고 주제(subjects) 및 범위(scopes)를 통해 적절한 메모리 구성을 사용하여 사용자 대화 기본 설정, 장치 진단 및 조직별 정보를 추적하고 활용합니다. |
| [OpenTelemetry](examples/OpenTelemetry.md)                                                                                          | Koog AI 에이전트에 OpenTelemetry 기반 트레이싱(tracing)을 추가합니다. 디버깅을 위해 스팬(spans)을 콘솔로 내보내고, Jaeger에서 보기 위해 OpenTelemetry Collector로 트레이스(traces)를 내보내는 방법을 알아봅니다. Docker 설정 및 문제 해결 가이드가 포함됩니다.                                             |
| [Planner](https://github.com/JetBrains/koog/tree/develop/examples/src/main/kotlin/ai/koog/agents/example/planner)                   | 병렬 및 순차 실행 노드를 사용하여 실행 트리를 구축하고, 복잡한 워크플로우를 위한 실행 계획을 동적으로 구성하는 작업 계획 시스템입니다.                                                                                                 |
| [PlaywrightMcp](examples/PlaywrightMcp.md)                                                                                          | Playwright MCP 및 Koog로 브라우저를 구동합니다. Playwright MCP 서버를 시작하고 SSE를 통해 연결하며, AI 에이전트가 자연어 명령을 통해 탐색, 쿠키 수락, UI 상호작용과 같은 웹 작업을 자동화하도록 합니다.                                      |
| [SimpleAPI](https://github.com/JetBrains/koog/tree/develop/examples/src/main/kotlin/ai/koog/agents/example/simpleapi)               | Koog를 시작하기 위한 간단한 API 패턴을 사용하여 채팅 에이전트 및 단일 실행 에이전트를 보여주는 기본 예제입니다.                                                                                                                                           |
| [StructuredData](https://github.com/JetBrains/koog/tree/develop/examples/src/main/kotlin/ai/koog/agents/example/structureddata)     | 복잡한 중첩 클래스, 다형성 및 일기 예보 예제를 포함한 JSON 기반 구조화된 데이터 출력을 보여주며, 에이전트 응답에서 타입이 지정된 데이터(typed data)를 작업하는 방법을 보여줍니다.                                                                                  |
| [SubgraphWithTask](https://github.com/JetBrains/koog/tree/develop/examples/src/main/kotlin/ai/koog/agents/example/subgraphwithtask) | 하위 그래프(subgraph) 전략을 사용하여 생성, 삭제 및 명령 실행을 포함한 파일 및 디렉터리 작업을 보여주는 프로젝트 생성 도구입니다.                                                                                                                |
| [Tone](https://github.com/JetBrains/koog/tree/develop/examples/src/main/kotlin/ai/koog/agents/example/tone)                         | 입력 텍스트에서 긍정적, 부정적 또는 중립적 톤을 식별하기 위해 특수 도구를 사용하는 텍스트 톤 분석 에이전트입니다. 감성 분석(sentiment analysis) 기능을 보여줍니다.                                                                                            |
| [UnityMcp](examples/UnityMcp.md)                                                                                                    | Unity MCP 서버 통합을 통해 AI 에이전트로 Unity 게임 개발을 구동합니다. stdio를 통해 Unity에 연결하고, 사용 가능한 도구를 검색하며, 에이전트가 자연어 명령을 통해 씬을 수정하고, 오브젝트를 배치하며, 게임 개발 작업을 실행하도록 합니다.        |
| [VaccumAgent](examples/VaccumAgent.md)                                                                                              | Koog 프레임워크를 사용하여 기본적인 반사 에이전트를 구현하는 방법입니다. 간단한 2셀 환경에서 환경 모델링, 도구 생성 및 자동화된 청소 작업을 위한 에이전트 동작을 다룹니다.                                                                         |
| [Weave](examples/Weave.md)                                                                                                          | OpenTelemetry (OTLP)를 사용하여 Koog 에이전트 트레이스(traces)를 W&B Weave로 추적하는 방법을 알아봅니다. 환경 변수를 설정하고 에이전트를 실행하며, 포괄적인 모니터링 및 디버깅을 위해 Weave UI에서 풍부한 트레이스(traces)를 확인합니다.                                                               |
